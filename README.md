# Sample Sinatra Webapp

A sample web application using ruby Sinatra.

# Getting started
## Native Ubuntu

```
$ sudo apt install rerun

$ bundle install

$ rerun ruby main.rb
````

## WSL / WSL2
`Rerun` does not work without the full Linux machinery.
```
$ bundle install

$ ruby main.rb
````

## All platforms
1. Point your web browser to http://localhost:4567
2. Run `./test`
